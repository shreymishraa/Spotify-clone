# Spotify Clone (Frontend)

A simple Spotify-like web player UI built with **HTML** and **CSS**.

> Educational project / UI clone. Not affiliated with or endorsed by Spotify.

## Features

- Sidebar navigation (Home / Search / Library)
- Cards layout for playlists/albums
- Sticky top navigation
- Bottom music player bar with:
  - Album section (cover + track info)
  - Player controls
  - Progress bar (range slider)

## Tech Stack

- HTML5
- CSS3
- Font Awesome (CDN)
- Google Fonts (Montserrat)

## How to Run Locally

1. Download/clone this repository.
2. Open `index.html` in your browser.

If you use VS Code, you can also run it with the **Live Server** extension.
